How to write tables in html

The tables below show how to set up and write tables into html. This whole webpage contains a number of html tables.

The table tags

Tag Sub-Tag Name Description
<table> Table tag This adds a table in html and needs to be used with table data and table rows
<tr> Table rows tag This will add rows to a table in html and to add row header this is the tag you would use
<td> Table data tag This will add rows of data to the table and is what you use to add data into rows rather than headers
<th> <th scope="col"> Titles tag This will add titles to a table the secondary tag shown will add a title to a column
<td> <td colspan="2"> Data spanning multiple columns tag This data will span over multiple columns it's like merge in excel
<td> <td rowspan="2"> Data spanning multiple rows tag This data will span over multiple rows it's like merge in excel
<tbody> Section the table tag This will section off parts of the table
<thead> Head of table tag This will section off the head of the table
<tfoot> Foot of table tag This will section off the foot of the table

Style can also be added to tables but this will be shown in the CSS section of the cheat sheet.
