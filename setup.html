HMTL Cheat Sheet

How to set up a html file

The tables below shows how I would set-up a html doc and what the tags mean. The first table will be compromised of the most important tags within html. Each of these are filled with smaller tags.

The key tags

Tag Sub-Tag Name Description
<!DOCTYPE html> Document type This simply lets the browser know that the code is to be written in html format
<html> HTML tag This will tell the webpage that everything inside this tag is html code
<meta> charset = "utf-8" Meta data part 1 This meta data does not appear on the page but lets the browser know what charcter set the html is using
<meta> name="name" content="description" Meta data part 2 This meta data will appear on the tab view or the google search for your website
<head> Head tag This data will not appear in your webpage but is data the web page will need such as links to style sheets etc
<body> Body tag This will contain all your data visible on the webpage

Once these tags have been set up we can start adding data and tags into the head section.

Tags within the head

The tags below are all entered between the opening and closing tag of the head.

Tag Sub-Tag Name Description
<link> href="style.css" rel="stylesheet" Link tag This will link your html document to your css document, but note the link tag can be used for general links as well(Goes within the head tag)
<title> Title tag This is simply the title of your web page but again it will not appear on the webpage(Goes within the head tag)

Then we can start to input data and tags within the body section. Everything that is within this section will appear visible on the web page.

Tags within the body

The table below is a general list of tags that can be used within the body seciton of the html document. (More complex tags can be found in the tables later on).

Tag Sub-Tag Name Description
<header> Header tag Similar to the head tag but it goes within the ody tag and does appear on your page as a header would appear on microsft word
<h1> <h2>, <h3>.... <h6> Heading tags These are your heading tags for the page and go down to h6 with h1 being the largest
<section> Section tags These specify new sections within your page
<div> Division tags Similar to section tags they divide up the page
<article> Article tags These specify new sections that are specifically article sections eg a block of text
<p> Paragraph tags Specify a new paragraph
<a> href="URL" Link tags These are link tags that can link any element using a URL(Note if you add target="_blank" the link opens in a new tab
<img> src="URL" alt="description of the image" Image tags Used to add an image into a webpage. If the image cannot be loaded the description is displayed
<ol> <li> Ordered list tags These create an ordered list which will be numbered
<ul> <li> Unordered list tags These create an unordered list which will be bullet pointed
